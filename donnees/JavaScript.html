Évolutions possibles

Un développeur en informatique aura beaucoup d'évolutions possibles tout au long de sa carrière.


Travailler dans tous les domaines : de nos jours, l'informatique se retrouve dans tous les domaines, de l'agriculture à l'aérospatiale.

Apprendre de nouveaux langages de code : à chaque utilisation de l'informatique son langage, les sites WEBs, les aplications mobiles ou les Intelligences Artificielles ont été codés avec des langages de code différents.
